# Sid Meier's Civilization VI — Türkçe Yama

Medeniyet VI oyunu için topluluk Türkçe çeviri yaması.

## İçerik

- **Orjinal/** — Oyunun İngilizce orijinal metin dosyaları (XML formatında)
- **Türkçe/** — Türkçeleştirilmiş metin dosyaları

### Kapsanan bölümler

- **Base/** — Ana oyun metinleri (medeniyetler, teknolojiler, yönetim şekilleri, birimler, binalar, harikalar, diplomasi, savaş, Civilopedia vb.)
- **DLC/** — Ek paket metinleri

Toplam ~400 XML dosyası, orijinal ve çeviri olmak üzere çift halde sunulmaktadır.

## Kullanım

1. Orijinal oyun dosyalarınızı yedekleyin.
2. `Türkçe/` klasöründeki XML dosyalarını oyunun `Base/Assets/Text/` ve `DLC/` klasörlerine kopyalayın.
3. Oyunda dil olarak İngilizce seçiliyken Türkçe metinleri göreceksiniz.

## Katkı

Çeviri hatalarını düzeltmek veya eksik çevirileri tamamlamak için PR gönderebilirsiniz.

## Lisans

Bu proje bir hayran yapımıdır. Tüm oyun içi metinlerin telif hakkı Firaxis Games / 2K Games'e aittir.
